Propiedades de Fuentes

CSS ofrece múltiples propiedades para controlar la apariencia del texto.

font-weight (Grosor del Texto)

Controla el grosor o peso de la fuente.

Valores disponibles:

Valor Descripción Equivalente Numérico
normal Peso normal de la letra 400
bold Texto en negrita 700
lighter Más delgado que el elemento padre -
bolder Más grueso que el elemento padre -
100-900 Valores numéricos (múltiplos de 100) 100 (delgado) a 900 (negro)

Ejemplos:

p {
font-weight: normal;  /* 400 */
}

h1 {
font-weight: bold;    /* 700 */
}

.delgado {
font-weight: 300;
}

.extra-bold {
font-weight: 800;
}

100 - Thin

300 - Light

400 - Normal

500 - Medium

700 - Bold

900 - Black

font-size (Tamaño del Texto)

Establece el tamaño de la letra del texto.

Unidades de Medida

Unidades Absolutas:

  • px (píxeles): Más común, tamaño fijo
  • pt (puntos): Usado principalmente en impresión
  • cm, mm, in: Medidas físicas

Unidades Relativas:

  • em: Relativo al tamaño de fuente del elemento padre
  • rem: Relativo al tamaño de fuente raíz (html)
  • %: Porcentaje relativo al padre
  • vw/vh: Relativo al viewport (ancho/alto de ventana)

Palabras Clave Absolutas:

  • xx-small, x-small, small
  • medium (valor por defecto)
  • large, x-large, xx-large

Palabras Clave Relativas:

  • larger: Más grande que el padre
  • smaller: Más pequeño que el padre

Ejemplos:

body {
font-size: 16px;  /* Tamaño base */
}

h1 {
font-size: 2.5em;  /* 2.5 veces el tamaño del padre */
}

p {
font-size: 1rem;   /* Igual al tamaño base */
}

.pequeño {
font-size: 12px;
}

.grande {
font-size: x-large;
}
Recomendación: Usar rem para mayor accesibilidad y facilidad de mantenimiento.

font-family (Tipo de Fuente)

Establece la fuente o tipo de letra con la que se muestra el texto.

Familias Genéricas

  • serif: Fuentes con remates (ej: Times New Roman)
  • sans-serif: Fuentes sin remates (ej: Arial, Helvetica)
  • monospace: Fuentes de ancho fijo (ej: Courier)
  • cursive: Fuentes que parecen escritas a mano
  • fantasy: Fuentes decorativas

Sintaxis con fuentes alternativas:

body {
font-family: Arial, Helvetica, sans-serif;
}

h1 {
font-family: "Times New Roman", Times, serif;
}

code {
font-family: "Courier New", Courier, monospace;
}

.elegante {
font-family: Georgia, "Palatino Linotype", serif;
}
Importante:
  • Listar fuentes en orden de prioridad
  • Siempre terminar con una familia genérica
  • Usar comillas para nombres con espacios

Fuentes Web (Google Fonts)

Importar desde Google Fonts:

<head>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@300;400;700&display=swap" 
        rel="stylesheet">
</head>

Usar en CSS:

body {
font-family: 'Roboto', sans-serif;
}

Colores en CSS

CSS ofrece múltiples formas de especificar colores.

Formatos de Color

1. Nombres de Colores

color: red;
background-color: blue;
border-color: green;

2. Hexadecimal (RGB)

color: #FF0000;        /* Rojo */
color: #00FF00;        /* Verde */
color: #0000FF;        /* Azul */
color: #F00;           /* Rojo (forma corta) */

3. RGB y RGBA

color: rgb(255, 0, 0);           /* Rojo */
color: rgb(0, 255, 0);           /* Verde */
color: rgba(0, 0, 255, 0.5);     /* Azul semi-transparente */

4. HSL y HSLA

color: hsl(0, 100%, 50%);        /* Rojo */
color: hsl(120, 100%, 50%);      /* Verde */
color: hsla(240, 100%, 50%, 0.5); /* Azul semi-transparente */

Propiedades de Fondo (Background)

background-color

body {
background-color: #f0f0f0;
}

.destacado {
background-color: rgba(52, 152, 219, 0.2);
}

background-image

body {
background-image: url('imagenes/fondo.jpg');
}

.patron {
background-image: url('patron.png');
}

background-repeat

div {
background-image: url('logo.png');
background-repeat: no-repeat;  /* no-repeat, repeat, repeat-x, repeat-y */
}

background-position

div {
background-position: center;     /* center, top, bottom, left, right */
background-position: 50% 50%;    /* Usando porcentajes */
background-position: 10px 20px;  /* Usando píxeles */
}

background-size

div {
background-size: cover;     /* Cubre todo el contenedor */
background-size: contain;   /* Se ajusta dentro del contenedor */
background-size: 100px 200px; /* Tamaño específico */
}

Propiedad Abreviada (Shorthand)

Puedes especificar todas las propiedades de fondo en una sola línea.

body {
background: #ffffff url("fondo.jpg") no-repeat center center;
}

/* Equivale a: */
body {
background-color: #ffffff;
background-image: url("fondo.jpg");
background-repeat: no-repeat;
background-position: center center;
}

Gradientes

Gradiente Lineal:

div {
background: linear-gradient(to right, #ff0000, #00ff00);
background: linear-gradient(45deg, #3498db, #8e44ad);
}

Gradiente Radial:

div {
background: radial-gradient(circle, #3498db, #2ecc71);
}

Bordes (Border)

Propiedades de Borde

div {
border-width: 2px;           /* Grosor */
border-style: solid;         /* Estilo */
border-color: #3498db;       /* Color */
}

/* Forma abreviada: */
div {
border: 2px solid #3498db;
}

Estilos de Borde

  • solid: Línea sólida
  • dashed: Línea discontinua
  • dotted: Línea punteada
  • double: Línea doble
  • groove, ridge, inset, outset: Efectos 3D
  • none: Sin borde

Bordes Individuales

div {
border-top: 3px solid red;
border-right: 2px dashed blue;
border-bottom: 1px dotted green;
border-left: 4px double orange;
}

border-radius (Esquinas Redondeadas)

Permite crear esquinas redondeadas en los elementos.

Sintaxis básica:

div {
border-radius: 10px;        /* Todas las esquinas */
border-radius: 50%;         /* Círculo perfecto */
}

/* Esquinas individuales: */
div {
border-radius: 10px 20px 30px 40px;
/* top-left, top-right, bottom-right, bottom-left */
}

Propiedades Individuales

div {
border-top-left-radius: 10px;
border-top-right-radius: 20px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 40px;
}

Otras Propiedades de Texto

text-align

p {
text-align: left;    /* left, right, center, justify */
}

text-decoration

a {
text-decoration: none;       /* Quitar subrayado */
}

.subrayado {
text-decoration: underline;
}

.tachado {
text-decoration: line-through;
}

text-transform

h1 {
text-transform: uppercase;   /* MAYÚSCULAS */
}

h2 {
text-transform: lowercase;   /* minúsculas */
}

h3 {
text-transform: capitalize;  /* Capitalize Each Word */
}

line-height

p {
line-height: 1.6;    /* Espaciado entre líneas */
}

letter-spacing y word-spacing

.espaciado {
letter-spacing: 2px;    /* Espacio entre letras */
word-spacing: 5px;      /* Espacio entre palabras */
}
