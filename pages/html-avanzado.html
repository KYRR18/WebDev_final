Multimedia en HTML5

Con HTML5, insertar archivos multimedia se volvió mucho más sencillo gracias a las etiquetas semánticas <audio> y <video>.

Etiqueta <audio>

Permite reproducir archivos de audio. Soporta tres formatos principales: MP3, OGG y WAV.

Atributos de <audio>

  • controls: Muestra controles de reproducción
  • autoplay: Reproduce automáticamente
  • loop: Repite el audio al terminar
  • muted: Inicia silenciado
  • preload: none (no precarga), auto (precarga), metadata (solo info)

Ejemplo recomendado (con múltiples formatos):

<audio controls loop preload="auto">
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    <source src="audio.wav" type="audio/wav">
    Tu navegador no soporta la etiqueta de audio.
</audio>
Importante: Usar <source> en lugar de src permite ofrecer múltiples formatos. El navegador usará el primero que soporte.

Etiqueta <video>

Permite insertar videos en la página. Similar a <audio> pero con atributos adicionales para dimensiones y vista previa.

Atributos específicos de <video>

  • width y height: Dimensiones del video
  • poster: Imagen de portada antes de reproducir
  • Comparte con audio: controls, autoplay, loop, muted, preload

Ejemplo completo:

<video width="800" height="600" 
       controls 
       poster="portada.jpg">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    <source src="video.ogg" type="video/ogg">
    Tu navegador no soporta video HTML5.
</video>

Enlaces y Navegación

Los hipervínculos se crean con la etiqueta <a> (anchor) y el atributo href.

Tipos de Referencias

1. Referencia Absoluta

Conduce a una ubicación externa al sitio. Debe incluir el protocolo y dominio completo.

<a href="https://www.ejemplo.com">Visitar Ejemplo</a>

2. Referencia Relativa

Conduce a un documento dentro del mismo sitio.

<!-- Archivo en la misma carpeta -->
<a href="pagina2.html">Ir a Página 2</a>

<!-- Archivo en subcarpeta -->
<a href="docs/documento.html">Ver Documento</a>

<!-- Archivo en carpeta superior -->
<a href="../index.html">Volver al Inicio</a>

3. Punto de Fijación (Anclas)

Conduce a un punto específico dentro de un documento.

Crear el ancla:

<h2 id="seccion1">Sección 1</h2>

Enlazar al ancla:

<!-- Mismo documento -->
<a href="#seccion1">Ir a Sección 1</a>

<!-- Otro documento -->
<a href="pagina.html#seccion1">Ver Sección 1</a>

Atributo target (Destino del Enlace)

Determina dónde se abrirá la página vinculada:

  • _blank: Abre en nueva ventana/pestaña
  • _self: Abre en la misma ventana (predeterminado)
  • _parent: Abre en el marco padre
  • _top: Abre en la ventana completa del navegador
<a href="https://www.buap.mx" target="_blank">
    Visitar BUAP (nueva ventana)
</a>

Tipos Especiales de Enlaces

Enlace de Correo Electrónico

<!-- Email simple -->
<a href="mailto:info@ejemplo.com">Enviar email</a>

<!-- Email con asunto prellenado -->
<a href="mailto:soporte@empresa.com?subject=Consulta">
    Contactar Soporte
</a>

Enlace de Descarga

<a href="documentos/manual.pdf" download>
    Descargar Manual PDF
</a>

Enlace Vacío

No abre ninguna página, útil para JavaScript o marcadores de posición.

<a href="#">Enlace placeholder</a>

Listas en HTML

Listas No Ordenadas <ul>

Listas con viñetas. Se pueden personalizar con el atributo type.

<ul>
    <li type="disc">Elemento con círculo relleno</li>
    <li type="circle">Elemento con círculo vacío</li>
    <li type="square">Elemento con cuadrado</li>
</ul>

Listas Ordenadas <ol>

Listas numeradas automáticamente.

<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>

Comenzar desde un número específico

<ol>
    <li value="20">Este será el número 20</li>
    <li>Este será el 21</li>
    <li>Este será el 22</li>
</ol>

Imágenes

Las imágenes se insertan con la etiqueta <img>, que no necesita etiqueta de cierre.

Sintaxis Básica

<img src="imagenes/foto.jpg" 
     alt="Descripción de la imagen"
     width="600" 
     height="400">

Atributos Importantes

  • src: Ruta de la imagen (requerido)
  • alt: Texto alternativo (accesibilidad y SEO)
  • width y height: Dimensiones en píxeles
  • title: Texto que aparece al pasar el mouse
Buena práctica: Siempre incluir el atributo alt para mejorar la accesibilidad y SEO. Si la imagen no se carga, se mostrará este texto.

Imágenes como Enlaces

Una imagen puede ser un hipervínculo encerrándola dentro de <a>.

<a href="pagina2.html">
    <img src="imagenes/boton.jpg" 
         alt="Ir a Página 2">
</a>

Formularios

Los formularios permiten recoger datos introducidos por el usuario.

Etiqueta <form>

Contiene todos los elementos del formulario.

Atributos principales:

  • action: URL donde se enviarán los datos
  • method: Método de envío (GET o POST)
  • enctype: Tipo de codificación de datos
<form action="procesar.php" method="post" enctype="multipart/form-data">
    <!-- Elementos del formulario -->
</form>

Diferencia entre GET y POST

  • GET: Para consultas que no modifican datos (búsquedas)
  • POST: Para enviar datos que modifican información (registros, comentarios)

Elementos de Entrada <input>

Los campos de entrada más comunes:

<!-- Campo de texto -->
<input type="text" name="nombre" placeholder="Tu nombre">

<!-- Email -->
<input type="email" name="email" required>

<!-- Contraseña -->
<input type="password" name="password">

<!-- Número -->
<input type="number" name="edad" min="18" max="100">

<!-- Checkbox -->
<input type="checkbox" name="acepto" id="acepto">
<label for="acepto">Acepto términos</label>

<!-- Radio buttons -->
<input type="radio" name="genero" value="m"> Masculino
<input type="radio" name="genero" value="f"> Femenino

<!-- Botón de envío -->
<input type="submit" value="Enviar">

Área de Texto <textarea>

Para textos largos de múltiples líneas.

<textarea name="comentario" 
          rows="5" 
          cols="50" 
          placeholder="Escribe tu comentario aquí">
</textarea>

Lista Desplegable <select>

<select name="pais">
    <option value="">Selecciona un país</option>
    <option value="mx">México</option>
    <option value="es">España</option>
    <option value="ar">Argentina</option>
</select>

Ejemplo de Formulario Completo

<form action="registro.php" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="mensaje">Mensaje:</label>
    <textarea id="mensaje" name="mensaje" rows="4"></textarea>
    
    <input type="submit" value="Enviar"> </form>
