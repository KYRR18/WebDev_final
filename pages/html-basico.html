1. Introducción a HTML

1.1. ¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para crear y estructurar el contenido de las páginas web. No es un lenguaje de programación; funciona mediante etiquetas y no tiene estructuras de control como bucles o condicionales.

1.2. Estructura Básica del Documento

Todo documento HTML se basa en una estructura fundamental:

  • La línea <!DOCTYPE html> declara la versión HTML5.
  • <html> es la etiqueta raíz.
  • <head> contiene información no visible para el usuario (como el título de la pestaña o metadatos).
  • <body> contiene todo el contenido visible de la página.
<!DOCTYPE html>
<html>
<head>
    <title>Título de la Ventana</title>
</head>
<body>
</body>
</html>

1.3. Etiquetas (Tags) y Atributos

  • Etiquetas: La base de HTML. La mayoría tienen una apertura (<etiqueta>) y cierre (</etiqueta>).
  • Etiquetas Vacías: Algunas etiquetas no necesitan cierre, como <br> (salto de línea) o <img> (imagen).
  • Atributos: Propiedades que se añaden a la etiqueta de apertura con el formato nombre="valor" para particularizar su apariencia o comportamiento.

Ejemplo: El atributo bgcolor en <body> cambia el color de fondo de toda la página. Los colores suelen usar el formato RGB Hexadecimal (e.g., #FF0000 para rojo).

2. Elementos de Estructura y Semántica (HTML5)

HTML5 introdujo etiquetas para dar sentido (semántica) a las diferentes partes del documento, lo que ayuda a los motores de búsqueda a entender el contenido.

2.1. Contenedores Genéricos

Etiqueta Descripción
<body> Define casi toda la página visible.
<span> Crea un bloque en línea dentro de un párrafo para aplicar un estilo diferente.
<iframe> Se usa para incrustar contenido de otra fuente (generalmente de otros sitios web).

2.2. Elementos Semánticos Estructurales

Estos elementos ayudan a definir las regiones clave de la página:

Etiqueta Propósito/Uso
<header> Especifica un encabezado de un documento o sección.
<nav> Agrupa enlaces de navegación principales (menú).
<section> Define secciones temáticas o funcionales (capítulos, introducción).
<article> Contenido independiente y autónomo (noticias, publicaciones de blog, comentarios de foros).
<aside> Contenido al margen del contenido principal (barras laterales, notas).
<footer> Define el pie de página; suele contener información de autor, copyright, y enlaces.

2.3. Elementos Semánticos de Contenido

  • <figure>: Especifica contenido como ilustraciones o diagramas. Su eliminación no debe afectar el flujo principal.
  • <figcaption>: Específica una denominación o título para la figura, y debe colocarse al inicio o al final del elemento <figure>.
  • <time>: Permite fijar una fecha y hora, usando el atributo datetime para el calendario gregoriano.

3. Enlaces (Hyperlinks)

La etiqueta <a> (ancla) se usa para insertar enlaces, y el atributo href especifica el destino.

3.1. Tipos de Referencia (href)

  • Absoluta: Conduce a una ubicación externa (debe incluir el dominio, e.g., https://...).
  • Relativa: Conduce a un documento dentro del mismo sitio (e.g., /acerca_de.html).
  • Punto de Fijación (Ancla): Conduce a un punto específico dentro de un documento usando #nombre_de_punto. Para crear el punto de fijación se usa el atributo name dentro de otra etiqueta <a>.
  • Correo Electrónico: La referencia es mailto:direcciondecorreo. Se puede prellenar el asunto: mailto:correo@ejemplo.com?subject=El asunto.
  • Vínculo Vacío: El valor de href es solo #, manteniendo el formato de enlace sin llevar a ninguna página.

3.2. Destino del Enlace (target)

El atributo target determina en qué ventana se abrirá el destino.

Valor de target Descripción
_self Opción predeterminada. Abre el documento en el mismo marco o ventana.
_blank Abre el documento vinculado en una ventana o pestaña nueva.
_top Abre el documento en la ventana completa del navegador.

4. Formato de Contenido y Caracteres

4.1. Salto de Línea y Espacios

  • Salto de Línea: La etiqueta vacía <br> se utiliza para forzar un salto de línea.
  • Espacios Múltiples: Para tener más de un espacio en blanco seguido, se usa el carácter &nbsp; (espacio duro), ya que HTML solo reconoce uno.

4.2. Caracteres Especiales (Entidades HTML)

Se usan entidades HTML para representar caracteres que tienen un significado especial (como < y >) o caracteres que no se visualizan correctamente.

Carácter Representación (Entidad)
< &lt;
> &gt;
á &aacute;
ñ &ntilde;

4.3. Texto Preformateado (<pre>)

Esta etiqueta le indica al navegador que debe respetar los espacios en blanco, los saltos de línea y la mayoría de caracteres especiales tal como están en el editor de texto.

Limitación: No puede incluir ciertas etiquetas, como <img> o <object>.

4.4. Comentarios

Son líneas de texto ignoradas por el navegador, usadas para documentar el código.

Sintaxis: <!-- Esto es un comentario -->

5. Imágenes y Multimedia

5.1. Imágenes (<img>)

La etiqueta <img> inserta una imagen y es vacía.

  • src: Obligatorio. Especifica la ruta de la imagen.
  • alt: Texto alternativo. Se muestra si la imagen no se puede visualizar o es utilizado por lectores de pantalla.
  • Imágenes como Enlaces: Para hacer una imagen clickeable, se encierra la etiqueta <img> dentro de <a>.
  • Gifs:Los navegadores actuales permiten al repreduccion de gifs

5.2. Audio (<audio>) y Video (<video>)

  • Permiten reproducir archivos multimedia de forma nativa en HTML5.
  • <source>: Se recomienda usar esta etiqueta dentro de <audio> o <video> para ofrecer múltiples formatos y mejorar la compatibilidad entre navegadores.
  • Atributos clave: controls (muestra controles de reproducción), autoplay (inicia automáticamente), y loop (repite el contenido).
  • El atributo poster (solo en <video>) define una imagen de preview para mostrar antes de que el video inicie.

6. Listas

6.1. Listas No Ordenadas (<ul>)

Se usan con <ul> (Unordered List) y cada punto se define con <li> (List Item).

El atributo type en el <li> permite cambiar el símbolo (e.g., disc, circle, square).

<ul>
    <li>Café</li>
    <li type="square">Té</li>
</ul>

6.2. Listas Ordenadas (<ol>)

Se usan con <ol> (Ordered List) y cada punto se define con <li>.

El atributo value en el <li> permite que la lista comience o salte a un número específico.

<ol>
    <li value="10">El punto diez</li>
</ol>

7. Formularios (<form>)

Los formularios (<form>) permiten recoger datos introducidos por el usuario.

7.1. Atributos de la Etiqueta <form>

Atributo Propósito
action Dirección a la que se enviarán los datos.
method El método de transferencia. get (datos visibles en la URL) o post (datos ocultos, usados para cambios).
enctype El modo en que se cifrará la información. Ejemplo: text/plain para envío por correo.

7.2. Elementos de Entrada

  • Área de Texto (<textarea>): Permite ingresar texto en múltiples líneas.
    • name: Nombre para identificar el campo.
    • rows / cols: Indican el alto y ancho visuales.
  • Elemento de Entrada (<input>): Etiqueta versátil cuya función cambia con el atributo type (e.g., type="text", type="submit").

Usos Prácticos: Ejemplos de Interacción

8.1. Construcción de una Estructura Navegable

Al combinar elementos semánticos de estructura con listas y enlaces, creamos menús de navegación robustos para todo el sitio.

Uso: Crear el menú principal de la página.

<nav>
    <ul>
        <li><a href="productos.html">Productos</a></li>
        <li><a href="#contacto">Contacto</a></li>
    </ul>
</nav>

8.2. Creación de Componentes Reutilizables

Combinando los contenedores autónomos con la inserción de imágenes, podemos crear bloques de contenido reutilizables como tarjetas de noticias.

Uso: Definir una publicación de blog completa y autónoma.

<section>
    <article>
        <h3>Las Novedades del Mes</h3>
        <img src="icono_novedad.png" alt="Icono de una estrella" width="50">
        <p>Este artículo podría ser movido a cualquier otra página sin perder su significado.</p>
    </article>
</section>

8.3. Haciendo Elementos Clickeables

Al encerrar una etiqueta de imagen (<img>) dentro de una etiqueta de enlace (<a>), convertimos un elemento visual estático en un hipervínculo interactivo.

Uso: Crear un botón o un icono de red social que lleva al usuario a un sitio externo.

<a href="http://www.buap.mx" target="_blank">
    <img src="logo_universidad.png" alt="Logo de la BUAP" width="100">
</a>
