Estructura Básica de HTML

HTML funciona a través de etiquetas (tags) que permiten la creación de contenido y su formato. Los documentos HTML se escriben en modo ASCII usando cualquier procesador de texto.

Caracteres Especiales

Los caracteres < y > son especiales. Para escribirlos como texto, usa &lt; y &gt;

Carácter Código Carácter Código
á &aacute; é &eacute;
í &iacute; ó &oacute;
ú &uacute; ñ &ntilde;
¿ &iquest; ¡ &iexcl;
  &nbsp; © &copy;

Etiqueta <body>

Define casi toda la página. Los cambios en los atributos del body afectan a toda la página.

<body bgcolor="#0300A0" background="imagen.jpg">
Contenido de la página
</body>

Comentarios

Líneas que el navegador no muestra, útiles para edición:

<!-- Esto es un comentario -->

Saltos de Línea

La etiqueta <br> genera un salto de línea.

La etiqueta <pre> permite texto preformateado que respeta espacios y saltos.

Elementos Semánticos HTML5

Elementos que ayudan a los navegadores a identificar el contenido de la página mediante etiquetas con significado.

<header>

Especifica un encabezado dentro de un documento o sección.

<nav>

Especifica un conjunto de vínculos de navegación.

<nav>
<ul>
    <li>Principal</li>
    <li>Fotos</li>
    <li>Contacto</li>
</ul>
</nav>

<section>

Define secciones en el documento como capítulos, encabezados, pies de página, etc.

<article>

Permite especificar contenido independiente o autónomo. Útil para foros, noticias, blogs, comentarios, etc.

<aside>

Define contenido al margen del contenido principal.

<footer>

Define el pie de página, que suele contener información del autor, copyright, enlaces, términos, contactos, etc.

<figure> y <figcaption>

Especifica contenido como ilustraciones o diagramas con su denominación.

<figure>
<figcaption>Logotipo HTML5</figcaption>
<img src="logo.png" alt="logotipo html5">
</figure>

<time>

Permite fijar una fecha y hora para recordatorio de eventos.

<time datetime="2025-11-26">26 de Noviembre</time>

Elementos Multimedia

<audio>

Permite reproducir archivos de audio (MP3, OGG, WAV).

Navegador MP3 WAV OGG
Chrome ✓ ✓ ✓
Firefox ✓ ✓ ✓
Safari ✓ ✓ ✗ 
<audio controls autoplay loop muted>
<source src="audio.mp3" type="audio/mpeg">
<source src="audio.ogg" type="audio/ogg">
Tu navegador no soporta audio
</audio>

Atributos de <audio>

  • autoplay: Reproduce automáticamente
  • controls: Muestra controles
  • loop: Repite el audio
  • muted: Inicia silenciado
  • preload: none, auto, metadata

<video>

Permite desplegar video (MP4, WebM, Ogg).

<video controls width="640" height="480" poster="preview.jpg">
<source src="video.mp4" type="video/mp4">
<source src="video.webm" type="video/webm">
Tu navegador no soporta video
</video>

<iframe>

Permite incrustar contenido de otra fuente.

<iframe src="https://ejemplo.com" width="600" height="400"></iframe>

Enlaces y Referencias

Etiqueta <a>

Crea hipervínculos que llevan a páginas o archivos.

<a href="https://ejemplo.com">Visitar sitio</a>

Tipos de Referencias

  • Absoluta: Conduce a ubicación externa (http://...)
  • Relativa: Dentro del mismo sitio
  • Punto de fijación: A un ancla dentro del documento

Atributo target

  • _blank: Abre en ventana nueva
  • _self: Misma ventana (predeterminado)
  • _parent: Marco padre
  • _top: Ventana completa
<a href="https://buap.mx" target="_blank">Visita BUAP</a>

Anclas

Crear puntos de fijación en el documento:

<a name="miancla"></a>
<a href="#miancla">Ir al ancla</a>

Enlaces Especiales

Correo electrónico:


<a href="mailto:correo@ejemplo.com">Enviar email</a>
<a href="mailto:correo@ejemplo.com?subject=Asunto">Email con asunto</a>

Descarga de archivos:

<a href="documento.pdf">Descargar PDF</a>

Vínculo vacío:

<a href="#">Vínculo vacío</a>

Listas

Listas No Ordenadas <ul>


<ul>
    <li type="disc">Punto negro</li>
    <li type="circle">Punto círculo</li>
    <li type="square">Punto cuadrado</li>
</ul>

Listas Ordenadas <ol>


<ol>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>

Iniciar en número específico:


<ol>
    <li value="20">Este será el 20</li>
    <li>Este será el 21</li>
    <li>Este será el 22</li>
</ol>

Imágenes

Etiqueta <img>

Inserta imágenes en la página. No necesita etiqueta de cierre.

<img src="imagenes/foto.jpg" alt="Descripción de la imagen">

Texto Alternativo (alt)

Se muestra cuando la imagen no puede cargarse.

<img src="gato.gif" alt="Imagen de un gato">

Enlaces con Imágenes

Combinar <a> con <img> para crear imágenes clicables:


<a href="pagina.html">
<img src="imagenes/foto.jpg" alt="Pintura Geométrica">
</a>

Formularios

Estructura Básica

Los formularios recopilan datos del usuario.


<form action="mailto:correo@servidor.com" method="post" enctype="text/plain">
<!-- Elementos del formulario -->
</form>

Atributos de <form>

  • action: Dirección de correo o programa que procesará el formulario
  • method: get o post
  • enctype: Modo de cifrado para envío

Área de Texto <textarea>


<textarea name="comentario" cols="30" rows="5">
Escribe aquí tu texto
</textarea>

Elementos de Entrada <input>

Diversos tipos de campos de entrada:


<input type="text" name="nombre" placeholder="Tu nombre">
<input type="email" name="email" placeholder="Tu email">
<input type="password" name="password">
<input type="submit" value="Enviar">
<input type="reset" value="Limpiar">
