¿Qué es CSS?

CSS (Cascade Style Sheets) es un lenguaje que trabaja junto con HTML para proveer estilos visuales a los elementos del documento, como tamaño, color, fondo, bordes, etc.

HTML5 y CSS3 son dos tecnologías diferentes que se complementan. CSS no forma parte de las especificaciones de HTML5.

Conceptos Básicos

CSS permite controlar la presentación visual de elementos HTML, separando el contenido (HTML) de la presentación (CSS). Esto facilita el mantenimiento y la actualización de sitios web.

Estructura y Modelo de Caja

Estructura de Bloque

Cada navegador implementa una estructura por defecto con dos tipos principales de elementos:

Elementos Block

Se posicionan uno sobre otro hacia abajo en la página. Ejemplos: <header>, <nav>, <div>, <footer>, <section>, <article>, <p>

Elementos Inline-Block

Se posicionan lado a lado en la misma línea, sin salto de línea a menos que no haya más espacio horizontal. Ejemplos: <span>, <img>, <a>

Modelo de Caja

Todas las reglas que especifican la disposición de una página web se denominan "Modelo de caja". Cada elemento HTML es una caja rectangular con:

  • Contenido: El área donde aparece el texto o las imágenes
  • Padding: Espacio entre el contenido y el borde (relleno interno)
  • Border: El borde que rodea el padding y el contenido
  • Margin: Espacio exterior que separa el elemento de otros (margen externo)

Métodos de Aplicación de CSS

1. Estilos en Línea

Forma simple, rudimentaria y poco práctica. Se aplican directamente en el elemento HTML usando el atributo style:

<p style="font-size: 20px; color: blue;">Esto es un párrafo</p>

Desventaja: Difícil de mantener y actualizar en sitios grandes.

2. Estilos Embebidos

Una mejor alternativa es insertar los estilos en la cabecera del documento usando la etiqueta <style>:

<head>
<style>
    p {
        font-size: 20px;
        color: blue;
    }
</style>
</head>

Ahora todas las etiquetas <p> se verán afectadas por estos estilos.

3. Archivos Externos (Recomendado)

Este método tiene más ventajas para facilitar la actualización de los estilos. Se implementa mediante el uso de la etiqueta <link>:

<head>
<link rel="stylesheet" type="text/css" href="css/estilos.css">
</head>

Ventajas:

  • Un solo archivo CSS puede estilizar múltiples páginas HTML
  • Fácil mantenimiento y actualización
  • Mejor organización del código
  • Caché del navegador mejora el rendimiento

Sintaxis de CSS

Un conjunto de reglas CSS consiste en un selector y un bloque de declaración:

h1 {
color: blue;
font-size: 12px;
}

Componentes:

  • Selector: h1 (apunta al elemento HTML que desea diseñar)
  • Propiedades: color, font-size (características a modificar)
  • Valores: blue, 12px (valores asignados a las propiedades)

El bloque de declaración contiene una o más declaraciones separadas por punto y coma. Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos. Una declaración de CSS siempre termina con punto y coma, y los bloques de declaración están rodeados de llaves.

Selectores CSS

Selector Universal (*)

Representa cada uno de los elementos del documento. Es útil cuando necesitamos establecer ciertas reglas básicas:

* {
margin: 0;
padding: 0;
}

Selector por Etiqueta

Selecciona todos los elementos de un tipo específico:

p {
color: #333;
line-height: 1.6;
}

Selector por ID (#)

Referencia un elemento específico mediante su atributo id. El ID debe ser único en el documento:

#articulo {
font-size: 20px;
}

p#aside {
color: red;
}
<p id="articulo">Texto del artículo</p>
<p id="aside">Texto del aside</p>

Notas importantes:

  • Es una referencia sumamente específica
  • No debe haber más de un elemento con el mismo ID
  • Normalmente se emplean IDs para elementos estructurales
  • Suele emplearse más en JavaScript

Selector por Clase (.)

Más flexible que el ID. Puede ser asignado a múltiples elementos HTML que comparten un diseño similar:

.destacado {
background: yellow;
font-weight: bold;
}

p.especial {
color: green;
}

Pseudo-clase :not()

Se emplea para realizar una negación. Aplica estilos a elementos que NO coinciden con el selector:

p:not(.classy) {
color: red;
}

*:not(p) {
margin: 0;
}

Tiene el mismo alcance que un selector universal.

Selector de Hijo Directo (>)

Selecciona elementos que son hijos directos de otro elemento:

div > p {
color: #990000;
}

Donde div es el elemento padre y p es el elemento hijo directo que se verá afectado.

Selector Adyacente (+)

Selecciona el elemento inmediatamente siguiente a otro elemento específico:

p.mitexto + p {
color: #990000;
}

El elemento <p> inmediatamente siguiente al elemento p con la clase mitexto se verá afectado por los estilos.

Selector de Hermanos (~)

Aplica estilos a todos los elementos hermanos que aparecen después de un elemento específico:

p.mitexto ~ p {
color: #990000;
}

El estilo será aplicado a todos los elementos <p> que son hermanos y se encuentran después del elemento <p> identificado con el valor mitexto en su atributo class.

Estilos para Fuentes

font-family

Establece la fuente o tipo de letra con la que se muestra el texto. Se pueden especificar fuentes alternativas separadas por comas. La lista tiene prioridad:

body {
font-family: 'Arial', 'Helvetica', sans-serif;
}

Los navegadores intentan mostrar el texto con la primera fuente, y si no está disponible, usan la segunda y así sucesivamente.

font-size

Establece el tamaño de letra del texto. Permite usar unidades absolutas y relativas (em, ex, px, %, in, mm, pt, pc):

h1 {
font-size: 2rem;
}

p {
font-size: 16px;
}

.pequeño {
font-size: smaller;
}

Palabras clave absolutas:

  • xx-small, x-small, small, medium, large, x-large, xx-large

Palabras clave relativas:

  • larger: Más grande que el elemento contenedor
  • smaller: Más pequeño que el elemento contenedor

font-weight

Controla el peso/grosor de la letra:

.normal {
font-weight: normal; /* o 400 */
}

.bold {
font-weight: bold; /* o 700 */
}

.ligero {
font-weight: 300;
}

.muy-grueso {
font-weight: 900;
}

Valores disponibles:

  • normal: Peso normal (igual que 400)
  • bold: Grueso ancho/negritas (igual que 700)
  • lighter: Menos grueso que el tipo del elemento padre
  • bolder: Más grueso que el tipo del elemento padre
  • 100-900: Pesos numéricos para mayor flexibilidad

Propiedades de Background

background-color

Define el color de fondo de un elemento:

body {
background-color: #ffffff;
}

background-image

Establece una imagen como fondo:

header {
background-image: url('imagen.jpg');
}

background-repeat

Controla la repetición de la imagen de fondo:

  • repeat: Repite en ambas direcciones (predeterminado)
  • repeat-x: Solo horizontal
  • repeat-y: Solo vertical
  • no-repeat: No repite la imagen

background-position

Define la posición inicial de la imagen de fondo:

div {
background-position: center center;
background-position: right top;
background-position: 50% 50%;
}

Propiedad Abreviada (background)

Para acortar el código, es posible especificar todas las propiedades de fondo en una sola propiedad:

body {
background: #ffffff url('imagen.jpg') no-repeat right top;
}

Bordes y Esquinas Redondeadas

border-radius

Con la propiedad CSS border-radius puede darle a cualquier elemento "esquinas redondeadas":

.caja-redondeada {
border-radius: 25px;
background: #ffffff;
padding: 20px;
}

Control Individual de Esquinas

La propiedad border-radius es en realidad una propiedad abreviada para:

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius
.esquinas-diferentes {
border-top-left-radius: 10px;
border-top-right-radius: 20px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 40px;
}

O usando la propiedad abreviada:

.esquinas {
border-radius: 10px 20px 30px 40px;
/* top-left | top-right | bottom-right | bottom-left */
}
