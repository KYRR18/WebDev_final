¿Qué es CSS?

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada).

Importante: HTML5 y CSS3 son dos tecnologías diferentes que se complementan. CSS no forma parte de las especificaciones de HTML5.

Definición Básica

CSS es un lenguaje que trabaja junto con HTML para proveer estilos visuales a los elementos del documento, como:

  • Tamaño de texto
  • Colores y fondos
  • Bordes y márgenes
  • Posicionamiento
  • Animaciones y transiciones

¿Por qué usar CSS?

  • Separación de contenido y presentación: HTML para estructura, CSS para diseño
  • Reutilización: Un archivo CSS puede aplicarse a múltiples páginas
  • Mantenimiento: Cambiar el diseño de todo un sitio editando un solo archivo
  • Carga más rápida: Los archivos CSS se almacenan en caché del navegador

Estructura de Bloque

Es la manera en la que un sitio web está dispuesto. Cada navegador implementa una estructura por defecto.

Elementos Block

Son elementos posicionados uno sobre otro hacia abajo en la página (por defecto en los navegadores).

Elementos de bloque comunes:

  • <div>
  • <p>
  • <h1> - <h6>
  • <header>, <nav>, <section>, <footer>
  • <ul>, <ol>, <li>

Elementos Inline-Block

Son posicionados lado a lado, uno al lado del otro en la misma línea, sin ningún salto de línea a menos que ya no haya más espacio horizontal.

Elementos inline comunes:

  • <span>
  • <a>
  • <strong>, <em>
  • <img>
  • <input>, <button>

Modelo de Caja (Box Model)

Todas las reglas que especifican la disposición en la que será organizada una página web se denomina "Modelo de caja".

Contenido
Padding (Relleno)
Border (Borde)
Margin (Margen)

Componentes del Modelo de Caja

  • Content (Contenido): El contenido real del elemento (texto, imagen, etc.)
  • Padding (Relleno): Espacio transparente entre el contenido y el borde
  • Border (Borde): Línea que rodea el padding y el contenido
  • Margin (Margen): Espacio transparente fuera del borde

Ejemplo de código:

div {
    width: 300px;          /* Ancho del contenido */
    height: 200px;         /* Alto del contenido */
    padding: 20px;         /* Relleno interno */
    border: 5px solid #000; /* Borde */
    margin: 10px;          /* Margen externo */
}
Nota: El ancho total del elemento = width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Formas de Aplicar CSS

1. Estilos en Línea (Inline)

Se aplican directamente en la etiqueta HTML usando el atributo style.

Ejemplo:

<p style="font-size: 20px; color: blue;">
    Este es un párrafo con estilo inline
</p>
Desventajas:
  • Difícil de mantener
  • No se reutiliza el código
  • Mezcla contenido con presentación
Solo usar en casos muy específicos.

2. Estilos Embebidos (Embedded)

Se insertan en la sección <head> del documento HTML dentro de la etiqueta <style>.

Ejemplo:

<head>
    <style>
        p {
            font-size: 20px;
            color: blue;
        }
        
        h1 {
            color: red;
            text-align: center;
        }
    </style>
</head>
Ventajas: Afecta a toda la página actual.
Desventajas: No se puede reutilizar en otras páginas.

3. Archivos Externos (External) ⭐ RECOMENDADO

Se crea un archivo .css separado y se vincula al HTML mediante la etiqueta <link>.

Archivo HTML:

<head>
    <link rel="stylesheet" 
          type="text/css" 
          href="css/estilos.css">
</head>

Archivo estilos.css:

p {
    font-size: 20px;
    color: blue;
}

h1 {
    color: red;
    text-align: center;
}
Ventajas:
  • ✅ Se puede reutilizar en múltiples páginas
  • ✅ Fácil de mantener
  • ✅ Separa completamente contenido de presentación
  • ✅ Mejora el rendimiento (caché del navegador)
Esta es la forma más profesional y recomendada.

Sintaxis de CSS

Un conjunto de reglas CSS consiste en un selector y un bloque de declaración.

Estructura básica:

selector {
    propiedad: valor;
    propiedad: valor;
}

Ejemplo completo con explicación:

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

Donde:

  • Selector: h1 (apunta al elemento HTML)
  • Propiedad: color, font-size, text-align
  • Valor: blue, 24px, center
  • Declaración: color: blue; (propiedad + valor)
Reglas importantes:
  • Cada declaración termina con punto y coma ;
  • Los bloques están rodeados de llaves { }
  • Propiedad y valor se separan con dos puntos :

Selectores en CSS

Los selectores determinan a qué elementos HTML se aplicarán los estilos.

1. Selector Universal (*)

Selecciona todos los elementos del documento.

* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

2. Selector de Elemento

Selecciona todos los elementos de un tipo específico.

p {
    color: black;
}

h1 {
    font-size: 32px;
}

a {
    text-decoration: none;
}

3. Selector por ID (#)

Selecciona un elemento único por su atributo id.

HTML:

<p id="articulo">Texto del artículo</p>

CSS:

#articulo {
    font-size: 20px;
    color: #333;
}
Importante: Un ID debe ser único en la página. No puede haber dos elementos con el mismo ID.

4. Selector por Clase (.)

Selecciona todos los elementos con una clase específica. Es más flexible que el ID.

HTML:

<p class="destacado">Párrafo 1</p>
<div class="destacado">Div 1</div>
<p class="destacado">Párrafo 2</p>

CSS:

.destacado {
    background-color: yellow;
    font-weight: bold;
}
Ventaja: Una clase puede aplicarse a múltiples elementos, y un elemento puede tener múltiples clases.

5. Selectores Combinados

Se pueden combinar selectores para mayor especificidad.

/* Párrafos con clase "especial" */
p.especial {
    color: red;
}

/* Divs con ID "contenedor" */
div#contenedor {
    width: 80%;
}

/* Múltiples selectores (mismo estilo) */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

6. Pseudo-clase :not()

Aplica estilos a todos los elementos excepto los especificados.

/* Todos los párrafos EXCEPTO los de clase "especial" */
p:not(.especial) {
    color: blue;
}

/* Todos los elementos EXCEPTO párrafos */
*:not(p) {
    margin: 0;
}

Selectores Avanzados

Selector de Hijo Directo (>)

Selecciona solo los hijos directos de un elemento.

HTML:

<div>
    <p class="texto">Este será afectado</p>
    <section>
        <p class="texto">Este NO será afectado</p>
    </section>
</div>

CSS:

div > p.texto {
    color: red;
}

Selector de Hermano Adyacente (+)

Selecciona el elemento inmediatamente siguiente a otro.

/* El primer <p> que viene después de un h2 */
h2 + p {
    font-weight: bold;
}

Selector de Hermanos Generales (~)

Selecciona todos los hermanos que vienen después de un elemento.

/* Todos los <p> que vienen después de h2 */
h2 ~ p {
    color: gray;
}
